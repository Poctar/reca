VIMware 


만약 한글을 치고 십다면
ex) shift+space바 동시에 누르기


vi 사용법 키 (명령 기능에서 사용가능 -> i 누르고 esc버튼눌러야 작동)

h : 방향키(뒤로 가는거)랑 똑같음 -> 위치 뒤로 한칸 이동
L : 방향키(앞으로 가는거)랑 똑같음 -> 위치 앞으로 한칸이동
J : 방향키(아래로 이동)랑 똑같음 -> 위치 아래로 이동
k : 방향키(위로 이동)랑 똑같음 -> 위치 위로 이동

x : 삭제(단어 한개를 지운다) ex) for example -> x를 누를때마다 한개씩 지워진다
			: 만약에 x를 한번 누르게 되면 결과 -> or example


i : vim에 들어왔을 때 누르게 되면 내가 입력을 할수 있게 해준다(입력 모드)
esc : 이것을 누르게 되면 명령모드로 전환 입력모드가 끝나게 된다
        ( 여기서는 for example를 입력모드에서 쳤으면 이 이상으로 칠수 없게 된다)

:w = 입력한것을 저장
:q = vim을 그대로 종료

w: 단어를 차레대로 띄어간다
b: 이전 단어들의 첫 글자로 이동한다

숫자0 : 커서가 있는 행의 시작으로 이동한다

$ : 커서가 있는 행의 끝으로 이동한다

gg : 첫 행으로 이동한다,   G: 마지막 행으로 이동한다 ,  dgg : 현재 커서 위치에서 문서 시작까지 삭제
							( 그 밑에 행은 사라지지 않는다)

dG: 현재 커서 위치에서 문서 끝까지 삭제 
( 위에 부분은 사라지지 않는다)

yy : 현재 커서가 가르키고 있는 행을 복사한다( 붙혀넣기는 p를 누르면 된다)
dd : 현재 커서가 가르키고 있는 행을 삭제
부분적으로 삭제하고 싶다면 esc를 누를 상태에서 v번 누르고 방향기로 하고 싶은 부분까지
하면 부분 삭제 가능!!


Rocky - 사용자 본인이 만든 계정
root = 처음부터 환경설정을 했을 떄 생성되는 계정 (신) - 아무 명령어나 해도 다 받아줌


u = 유저(사용자) , g (그룹) - 어딘가에 속해있음, o (그외)
r w x 다가능          w x 가능                              x 가능



ex) rocky $,  root #
-> 이 두개의 차이점은 rocky는 본인의 도어락 안에 있는 것은 수정이 가능하지만 다른 사람이
와서 수정하려고 하면 불가능  
하지만 root는 신이기 때문에 가능하다!
